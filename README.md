User Management App A React-based CRUD application for managing users, utilizing the JSONPlaceholder API for simulated data operations. This project demonstrates full-stack frontend development with responsive design and error handling.

Features:

User Listing: Displays users in a table format with name, email, and phone details.

Create User: Form to add new users via POST request to the API.

Update User: Edit existing users with pre-filled forms and PUT requests.

Delete User: Remove users with DELETE requests and confirmation prompts.

Detailed Views:

Individual user pages with additional information.

Responsive Design: Optimized for desktop and mobile devices.

Error Handling: User notifications for API failures.

Loading States: Spinner indicators during data fetching.

Tech Stack: React (functional components and hooks)

React Router for navigation

Axios for API interactions

CSS for styling and responsiveness

JSONPlaceholder API for mock data

Installation Clone the repository: git clone https://github.com/Jaspalnegi/User-Management-App.git Navigate to the project directory: cd user-management-app

Install dependencies: npm install

Start the development server: npm start
